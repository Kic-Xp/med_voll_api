Aqui é apenas um pequeno passo a passo de como iniciar o projeto

java -Dspring.profiles.active=prod -DDATASOURCE_URL=jdbc:mysql://localhost:3306/voll.med_api -DDATASOURCE_USERNAME=root -DDATASOURCE_PASSWORD= -jar api-0.0.1-SNAPSHOT.jar 

OBS: tira os paramentros -D e -DDATASOURCE caso voce tenha as variaveis de ambiente configuradas.

depois de rodar digita no navegar a URL: http://localhost:8080/swagger-ui.html e testa aí, só lembra de 
colocar o bearer token no authorize, faz o login no /login.
